# Notes for using fugitive plugin NVIM

## Basic stuff

### Staging

Add the currently open file git, i.e. stage current file:

```NVIM
:Gwrite
```

### Commit

Commit using the simple git Commit command:

```NVIM
:Git commit -m "message"
```
